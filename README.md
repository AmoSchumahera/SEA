Това е цялостно десктоп приложение, създадено като дипломен проект. Проектът представлява модерна версия на класическата логическа игра Minesweeper, надградена със система за акаунти, сигурна база данни, динамични класации и персонализиран потребителски интерфейс.

Приложението е изградено стриктно по **MVVM (Model-View-ViewModel)** архитектурния шаблон, което гарантира пълно разделяне на логиката от потребителския интерфейс.

## ✨ Основни функционалности

- **Система за потребители:** Сигурна регистрация и вход. Паролите не се пазят в чист текст, а се хешират чрез `BCrypt.Net-Next`.
- **Глобални класации (Leaderboards):** Седмична, Месечна и All-Time класация, показващи Топ 10 играчи. Данните се извличат от SQL база данни, като се сортират автоматично първо по брой мини, а след това по най-кратко време. Вградена е визуална система със златни, сребърни и бронзови медали.
- **Динамичен двигател на играта:** - Персонализиран брой мини (до 216).
  - Защита от първи клик (първият ход никога не е мина).
  - Рекурсивен *Flood-Fill алгоритъм* за мигновено отваряне на празните полета.
- **Модерен UX/UI:** - Динамична промяна на мащаба на дъската (Zoom Slider).
  - Поддръжка на 3 визуални теми (**Светла**, **Тъмна / Dark Mode** и **Ретро / Windows 95**), които се сменят в реално време чрез XAML DataTriggers, без презареждане на играта.
- **Мултимедия:** Вградени аудио ефекти при победа и експлозия.

## 🛠 Технологичен стек

- **Език:** C#
- **Фреймуърк:** .NET 10.0
- **Потребителски интерфейс:** WPF (Windows Presentation Foundation)
- **Архитектура:** MVVM (Model-View-ViewModel)
- **База данни:** Microsoft SQL Server (LocalDB)
- **ORM:** Entity Framework Core
- **Сигурност:** BCrypt.Net-Next (за хеширане на пароли)

## 🚀 Инструкции за стартиране (Локално)

За да стартирате проекта на вашия компютър, следвайте тези стъпки:

### 1. Изисквания
- Инсталиран **Visual Studio 2022** (с включен пакет *.NET desktop development*).
- Инсталиран **.NET 10.0 SDK**.
- **SQL Server Express** или **LocalDB** (включен по подразбиране във Visual Studio).

### 2. Клониране или изтегляне на проекта
Отворете `Minesweeper.Game.sln` чрез Visual Studio.
